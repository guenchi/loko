# Loko Scheme

Loko Scheme is an implementation of the algorithmic programming
language R6RS Scheme. It runs on Linux amd64 and on bare amd64
hardware.

## Current status

* Giant hack.
* Fun to look at.
* Self-hosting.

## Contact

https://gitlab.com/weinholt/loko-scheme/issues and `#loko` on Freenode.
